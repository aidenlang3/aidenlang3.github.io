Badger+ 2024 MadData Hackathon Winner

Badger+ is an application designed exclusively for UW Madison students, providing personalized academic advice and information. By inputting their Canvas API token, users can receive tailored guidance on their academics, including course recommendations, study tips, and scheduling assistance.

Additionally, Badger+ boasts an extensive database of UW Madison clubs, allowing students to explore and learn more about various extracurricular activities on campus.

AI Lung Imaging: Disease Recognition

A machine learning tool that accepts a lungs x-ray and can diagnose COVID-19, Bronchitis, Viral Pneumonia, and Bacterial Pneumonia with 98.6% accuracy.

I used Google Cloud Vertex AI for image classifcation and built a PyTorch convolutional network to differentiate between each disease classification. I used PyTorch's built in data augmentation features to improve model accuracy.

  • PyTorch
  • Google Cloud Vertex
  • Convolutional Neural Networks
